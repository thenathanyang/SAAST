The program in isbn.py takes in an ISBN-10 code from standard input, calculates 
the check digit, and determines whether or not the entered ISBN-10 code is valid
or invalid. 

Some valid ISBN-10 codes are:
	123456789X
	0136091814

An invalid ISBN-10 code is:
	0136091812

To run this program, enter this command in the command line:
	python isbn.py
